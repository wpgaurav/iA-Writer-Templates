### For macOS:
```bash
cp -r AcademicPaper.iatemplate ~/Library/Application\ Support/iA\ Writer/Templates/
cp -r GauravTiwari.iatemplate ~/Library/Application\ Support/iA\ Writer/Templates/
```

Then restart iA Writer and select from the template menu.

### For Windows:
Copy the `.iatemplate` folders to your iA Writer installation's Templates directory.

### For iOS/iPad:
The templates will sync via iCloud if using macOS. Or import through the template manager.

### Academic Paper Template

  - 900px max-width for optimal reading
  - Red accent color (#e74c3c)
  - Mobile-responsive design
