# Template Comparison

## Side-by-Side Feature Comparison

| Feature | Academic Paper | Gaurav Tiwari |
|---------|-----------------|-----------------|
| **Primary Font** | STIX Two Text | Inter |
| **Font Style** | Serif (Academic) | Sans-serif (Modern) |
| **Base Font Size** | 11pt | 1rem (16px) |
| **Line Height** | 1.8 | 1.7 |
| **Text Alignment** | Justified | Left-aligned |
| **Max Width** | Variable (1in margins) | 900px |
| **Color Scheme** | Neutral/Professional | Red accent (#e74c3c) |
| **Headings** | Hierarchical with borders | H2 with red left border |
| **Background** | White | White with gradient on title |
| **Use Case** | Academic/Technical | Web/Blog Content |
| **Responsive** | Print-optimized | Mobile-responsive |
| **Code Blocks** | Subtle styling | Modern with borders |
| **Tables** | Professional format | Contemporary with hover |
| **Mathematical Support** | Excellent (STIX) | Good (system default) |

## Quick Selection Guide

### Choose Academic Paper if you're writing:
- ✓ Academic papers or research articles
- ✓ Mathematical or scientific documents
- ✓ Technical documentation
- ✓ Formal professional documents
- ✓ Papers for publication

### Choose Gaurav Tiwari if you're writing:
- ✓ Blog posts or articles
- ✓ Web-based content
- ✓ Portfolio pieces
- ✓ Personal documentation
- ✓ Creative writing
- ✓ Modern web-style documents

## Design Philosophy

### Academic Paper Template
- **Philosophy:** Professional, traditional academic publishing standards
- **Inspiration:** Mathematical academic papers
- **Font Choice:** STIX Two Text - designed specifically for academic and scientific publishing
- **Goal:** Make serious writing look authoritative and scholarly

### Gaurav Tiwari Template
- **Philosophy:** Modern web design meets elegant typography
- **Inspiration:** Contemporary personal websites and blogs
- **Font Choice:** Inter - modern, system-friendly, web-native
- **Goal:** Make writing look contemporary and accessible

## CSS Architecture

Both templates use modern CSS with:
- CSS Custom Properties (variables) for easy theming
- Mobile-responsive media queries
- Print-optimized styles
- Semantic HTML structure
- Accessibility-focused styling

## Customization Difficulty

Both templates are designed for easy customization:
- **Beginner-friendly:** Change colors in `:root` variables
- **Intermediate:** Modify spacing and sizes
- **Advanced:** Restructure HTML layouts in template files

## Performance

Both templates:
- Use Google Fonts (web-safe)
- Minimal CSS footprint
- No JavaScript dependencies
- Fast loading and rendering
- Print-optimized

## Browser/Platform Support

Both templates work on:
- macOS (iA Writer)
- Windows (iA Writer)
- iOS/iPad (iA Writer)
- Web (via iA Writer Web)
- PDF export (both templates)
- Markdown export (both templates)
