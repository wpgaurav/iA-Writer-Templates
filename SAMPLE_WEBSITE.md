# Sample Content for Gaurav Tiwari Website Template

This is sample content demonstrating the Gaurav Tiwari website-inspired template.

## About This Template

The Gaurav Tiwari template brings a modern, web-inspired aesthetic to iA Writer. It features clean typography with the Inter font and contemporary design elements.

### Key Features

This template includes:

- Modern sans-serif typography (Inter font)
- Contemporary color scheme with accent red
- Generous whitespace and breathing room
- Mobile-responsive design
- Beautiful gradient on title pages
- Hover effects and interactive elements
- Professional code block styling

## Typography Examples
